## 📋 ในชีต (`index.html`) มีอะไรบ้าง

- **Breakpoint เปลี่ยนของตามเลเวล** (★ = จุดสำคัญ: Lv.16 / 33 / 45 / 70)
- ตาราง Base เกราะเต็ม (ค่า Evasion ต่อชิ้น) + **ชนิดเกราะทั้ง 6 แบบ**
- อาวุธ — หอก 2 set (Speed / Damage) + stat priority
- **Movement Speed รองเท้า** ตาม item level
- **เครื่องประดับ + Charm** — ilvl breakpoint ของมอด Life + แผนเปลี่ยนพร้อมเกราะ
- กฎเหล็ก HC: **Res 75% + Life มาก่อน Evasion เสมอ**
