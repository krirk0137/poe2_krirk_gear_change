4ลำดับลงทุนต่อชิ้น

1
Body Armour — สำคัญสุด ให้ evasion เยอะสุด (ถึง 519) + เป็นแหล่ง Deflection หลัก · อัปทุก breakpoint ที่ทำได้

3

4
Gloves เลือกจาก Res / Life / Attack Speed · evasion base น้อยมาก (สูงสุด 170) แทบไม่ต้องสน

5อาวุธ — หอก 2 set (หัวใจของ build)

build นี้ต้องใช้ หอก 2 อันคนละ stat สลับด้วยระบบ Weapon Swap — ไม่ใช่ทางเลือก แต่เป็นกลไกหลัก:

Weapon Set I

Speed → Whirling Slash

  • เน้น % Attack Speed สูงสุด
  • off-hand ต้องว่าง (เพื่อ Dance with Death)
  • ปั่น Whirling Slash ให้ไว + เดินเร็ว
Weapon Set II

Damage → Twister

  • เน้น flat Physical Damage สูง + Crit
  • ใส่ Twister / skill ปล่อยดาเมจไว้ set นี้
  • สลับมา set นี้ตอนจะ burst

ลำดับ stat บนหอก (ทั้ง 2 set ต้องมี Life + Res ผ่านของชิ้นอื่น):

ลำดับStatทำไม
1Physical Damageสเกลดาเมจหลักทั้ง build
2Attack Speedสำคัญสุดบน Set 1
3Mana Leechจำเป็นใน HC — กันมานาหมดกลาง pack
4Crit Chance + Crit Damageสเกล Set 2 (Twister)
5Accuracyกันพลาดตอน crit build

7กฎเหล็ก HC (เหนือทุก breakpoint)

Res cap 75% + Life มาก่อนเสมอ. ค่า Evasion base เป็นเรื่องรอง — ถ้าของชิ้นใหม่ทำให้ Resistance หลุด 75% หรือ Life ลด อย่าใส่เด็ดขาด ไม่ว่าจะถึง breakpoint หรือไม่ก็ตาม โดยเฉพาะช่วง campaign ที่ res ตกง่าย สาเหตุการตายอันดับต้นๆ ใน Hardcore คือ res หลุดตอนเจอ elemental burst
