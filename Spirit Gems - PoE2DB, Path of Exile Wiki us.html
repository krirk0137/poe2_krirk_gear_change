Spirit Gems

An Uncut Spirit Gem is a currency item. It is a type of uncut gem that can be used to engrave a persistent buff skill. It is a type of uncut gem.

Item acquisition

Uncut Spirit Gem has restrictions on where or how it can drop.

Can be obtained as a quest reward.
